# 🧠 NeuroBridge AI - MRI Classification

Brain tumor classification system using ResNet18 deep learning model.

## Features
- 4-class classification: Glioma, Meningioma, Pituitary, No Tumor
- 96.81% test accuracy
- Grad-CAM explainability
- Real-time inference

## Usage
Upload an MRI image and get instant classification results!

## Model Details
- Architecture: ResNet18
- Training: 13,127 images, 30 epochs
- Performance: 96.81% accuracy on 3,670 test samples
